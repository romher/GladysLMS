# GladysLMS
An interface with Logitech Media Server to use basic functions in Gladys

Note : This is a prototype. This project is not documented and not tested. Use it carefully.

# Manual Installation
(English version)
- Create a new parameter in gladys named "IP_LMS" with your LMS server IP adress and port (x.x.x.x:xxxx).
 
- Create a new directory GladysLMS in your modules directory (mkdir *USER*/gladys/api/hooks/GladysLMS , replace *USER* by your Gladys installation directory localization).

- Copy the index.js file and the lib directory in the new GladysLMS directory.

- Restart Gladys.

(Version française)
- Créer un nouveau paramètre "IP_LMS" dans Gladys avec l'IP et le port de votre serveur LMS (x.x.x.x:xxxx).
 
- Faire un nouveau dossier GladysLMS dans le dossier des modules Gladys (mkdir *USER*/gladys/api/hooks/GladysLMS , remplacer *USER* par l'emplacement de votre dossier d'installation de Gladys).

- Copier le fichier index.js et le dossier lib dans le nouveau dossier GladysLMS.

- Redémarrer Gladys.
